# Пакет работы с математическими функциями

Данный пакет позволяет реализовать полноценную работу с математическими функциями

## В данной версии представлены:
- Токенизатор математических выражений
- Приближённые вычисления математических уравнений методом дихотомии (для функций одной переменной и констант)
- Вычисление значений функций n переменных
- Удобный CLI-интерфейс
- UX: user-friendly опыт использования
